Depression AI


How to run the Application

1.) Ensure that you have Xampp Install.
2.) Ensure that You have php.
3.) Edit depression.php to enter Azure Facial Recognition API KEY, Azure Text-Analytics(Sentimental AI) API key
 and api url endpoint respectively.
4.) Call up the application on Browser http://localhost/depressionAI
